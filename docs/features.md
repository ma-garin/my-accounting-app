# 機能仕様

## メイン画面 (index.html)

### ラウンジダークUI

- 暗い店内向けのチャコール基調、シャンパンゴールド、ボルドー警告、ミント安全色を使用
- 現在見込み、次セットまで、予算残り、退店予定時見込みを状況カードで表示
- 下部タブの上に片手操作用のクイック追加バーを表示
- 個人メモ、キャスト名、満足度は保存メモ詳細内に格納

### タイマー管理

- 開始、一時停止、終了、時間リセットに対応
- セット時間は選択中の店舗プロファイルから取得
- 残り5分未満で残時間と状態チップを danger 表示
- `timer_state` と `active_visit_session` に状態を保存し、24時間以内の当日セッションを自動復元

### ドリンク管理

- 店舗プロファイルに登録されたドリンク価格を使用
- Lドリンク、メガドリンク、SHOTは固定クイック追加バーから即時追加可能
- 数量変更後は数秒間 Undo トーストを表示
- カスタムドリンクは最大5件まで追加でき、当日セッションに保存される

### 誤操作対策

- 会計全リセット、履歴全削除、シミュレーションリセットはボトムシート確認で実行
- リセット系操作は通常の追加操作から距離を取り、危険色を抑えめに表示

### 予算・退店判断

- 予算設定時に進捗バー、予算残り、予算80%超過、予算超過を表示
- 残予算で追加可能な主要ドリンク数を表示
- あと何セット、あと何分が安全圏かを表示
- 退店予定、閉店時刻、終電/次予定のうち最も近い制約を警告表示
- 退店予定時の追加セット数、追加料金、見込み合計を表示

### 会計サマリー

- セット合計、ドリンク合計、小計、サービス料、消費税、合計金額を表示
- サービス料率、税率、初回/延長セット料金は店舗プロファイルから取得
- 合計金額は100円単位で四捨五入

### 会計履歴

- `accounting_history` に最大50件保存
- 保存時に店名、キャスト名、指名有無、満足度、メモ、実請求額を任意保存
- 保存済み履歴には当時の料金スナップショットを保持
- 予測額と実請求額の差額を表示
- 履歴から複製して開始、シミュレーションへ転用、実請求額編集が可能
- 旧形式の履歴は読み込み時に新形式へマイグレーション

## シミュレーション画面 (simulation.html)

- 店舗プロファイルの料金で事前試算
- 滞在時間、ドリンク、シャンパン、カスタムドリンクを入力
- 「軽め」「通常」「長時間」「シャンパンあり」のテンプレートを適用可能
- シミュレーション結果を `saved_simulations` に保存
- 直近の実会計履歴との差額を表示
- 履歴から転用された内容を初期値として読み込み可能

## メニュー画面 (menu.html)

- 店舗プロファイルのドリンク/シャンパン一覧を表示
- メニュー検索、価格帯フィルタ、カテゴリ折りたたみに対応
- シャンパン価格を押すと、当日セッションの現在会計に追加した場合の概算総額を表示
- 各メニューには任意メモ欄を表示

## 店舗設定画面 (settings.html)

- 店舗名、セット時間、初回セット料金、延長セット料金、サービス料、税率を編集
- ドリンク価格、シャンパン価格を編集
- 既存プロファイルを複製して新店舗として保存
- アクティブ店舗を選択
- 料金変更履歴を最大20件保持

## PWA 対応

- `manifest.json` によるホーム画面追加
- Service Worker によるオフラインキャッシュ
- 更新検知時に `v2.16` 更新バナーと更新ボタンを表示
- 更新後は Service Worker の `controllerchange` で自動リロード
