# 機能仕様

## メイン画面 (index.html)

### タイマー管理

- **開始**: 現在時刻を `tsStart` に記録し、1秒ごとに `tick()` を実行
- **一時停止**: 経過時間を `tsPaused` に蓄積、タイマー停止
- **終了**: タイマー停止・状態リセット（セット数も0に）
- **リセット**: 終了 + ドリンク数量リセット（confirm ダイアログあり）
- セット切替は40分単位で自動カウントアップ
- 残り5分未満で残時間が赤く点滅
- ページリロードに備え `localStorage` にタイマー状態を保存

### ドリンク管理

**メインドリンク**（カードグリッド表示）
| ID | 名前 | 単価 |
|----|------|------|
| ldrink | Lドリンク | ¥1,700 |
| megadrink | メガドリンク | ¥3,700 |

**その他ドリンク**（アコーディオン内）
| ID | 名前 | 単価 |
|----|------|------|
| shot | SHOT | ¥1,500 |
| guestshot | ゲストSHOT | ¥1,000 |
| karaoke | カラオケ5曲 | ¥1,000 |

**カスタムドリンク**
- 最大5件まで追加可能
- 名前・金額をその場で入力
- リセットまで保持（localStorage には保存されない）

### 予算管理

- 予算を設定すると進捗バーが表示（0% → 80% → 100% で色変化）
- 超過時に赤いアラート表示
- 残予算でオーダー可能な各ドリンク数をリアルタイム表示
  - 計算対象: Lドリンク・メガドリンク・SHOT・ゲストSHOT
  - 税込単価（税 21% 相当を加算し100円単位切り上げ）で除算
- 予算はページリロードで引き継がない（毎入店で入力する設計）

### 退店逆算

- 退店予定時刻（HH:MM）を入力すると自動計算
- 退店までの残り時間
- 追加セット数（現セット残り後の追加分）
- 追加セット料金（税込、`SET_B × extra × 1.21` を100円丸め）
- 次のセット切替まで（残り5分未満で danger 表示）

### 会計サマリー

- セット合計 / ドリンク合計 / 小計 / サービス料 / 消費税 / 合計金額
- 合計金額は100円単位で四捨五入

### 訪問比較

- 今回金額 / 過去平均 / 前回比を並列表示
- 前回比: 増加は赤（up）、減少は緑（down）

### 会計履歴

- 「会計保存」ボタンで localStorage に保存（最大50件、古いものを削除）
- 新しい順で表示
- 全削除機能あり

## シミュレーション画面 (simulation.html)

訪問前の試算ツール。以下を入力して合計金額を試算：
- 滞在時間（セット数を自動計算）
- ドリンク種別・本数
- シャンパン種別・本数

---

## メニュー画面 (menu.html)

ドリンク・シャンパンのメニューと価格一覧を表示。静的表示のみ。

---

## PWA 対応

- `manifest.json` によるホーム画面追加（iOS/Android）
- Service Worker によるオフライン動作
- iOS: `apple-mobile-web-app-capable` メタタグ対応
- 更新検知時に画面上部に「今すぐ更新」バナーを表示
